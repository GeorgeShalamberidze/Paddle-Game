# Paddle-Game

## Built with: 
 - Webpack
 - Typescript
 - PixiJS

## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/GeorgeShalamberidze/Paddle-Game.git
   ```
2. Install packages
   ```sh
   npm install
   ```
3. Start 
   ```sh
   npm run serve
   ```
